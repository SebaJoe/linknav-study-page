# Instructions for the study

## Before Reading

Before you start reading your selected paper using LinkNav. Please watch the video above to familiarize yourself with the interface and its features. When you start your reading, you will need to set aside around an hour of your time to read the paper thoroughly and complete the survey with thoughtful feedback.

### Interface Prerequisites

Before you start using the interface, ***disable all adblockers***. We will need to track what you click on the interface page while you read, and we will not be able to collect this data if you have any adblocking software enabled. 

The interface should work on most browsers. It should definitely work on Firefox. Please let the study conductor know if you encounter any technical difficulties. 

## During Reading

You will need to read the paper you selected deeply. Imagine as if you are preparing to present this paper for a class or seminar group. Use that mindset to engage closely with the paper via the interface. In some cases, figures, tables, or citations may be difficult to read. If you find them distorted, you can reference the original PDF to clarify any missing details. 

## After Reading

After reading, please complete the post-reading questionnaire. Answer thoughtfully to the questions provided. This is your forum to provide any and all feedback about the interface!

***If you are done reading the instructions and watching the video, please click the checkbox below to get access to the LinkNav Interface and the Post-Reading Questionnaire.***
